# Autotype Engine

A predictive text engine built from scratch in Python.

## Current Features
- Dictionary-based autocomplete
- Custom prefix matching algorithm (without using Python's built-in startswith())
- 5 suggestion limiting

## Roadmap
- [x] Basic prefix matching <<----- we're currently at this stage
- [ ] Trie-based search
- [ ] Word ranking
- [ ] Learning from user selections
- [ ] Typo correction

## Example

Input:
he

Output:
hello
help
hero
helicopter
hey
